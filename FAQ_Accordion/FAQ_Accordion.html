What is an accordion? +
An accordion is a UI pattern where sections expand or collapse.
How does it work? +
It opens on click and closes when another is opened.
Why do we use accordions? +
To reduce clutter and show content only when needed.
Can an accordion contain icons or arrows? +
Yes. Many accordions include a + / – icon or a chevron arrow that rotates when opened.
Are FAQ accordions good for mobile? +
Yes! They are highly mobile-friendly because they reduce long blocks of text and show only what the user taps.
Does an accordion affect SEO? +
No, Google can still read hidden content, so SEO is not impacted as long as the FAQ is relevant.
